# 📦 离线打包教程 - 给老师/同学

## 方法一：直接发送 dist 文件夹（最简单）

### 步骤 1：构建项目
```bash
cd ~/桌面/company-website
npm run build
```

### 步骤 2：压缩 dist 文件夹
```bash
cd ~/桌面
zip -r TF-SSD-Website.zip company-website/dist
```

### 步骤 3：发送文件
把 `TF-SSD-Website.zip` 发给老师/同学

### 步骤 4：对方使用方法
1. 解压 `TF-SSD-Website.zip`
2. 打开终端，进入解压后的文件夹
3. 运行以下命令启动服务器：

```bash
# 方法 A：使用 Python（Mac/Linux 自带）
cd dist
python3 -m http.server 8080

# 方法 B：使用 Node.js（需要安装 Node）
npx serve dist -l 8080
```

4. 打开浏览器访问：`http://localhost:8080`

---

## 方法二：打包整个项目（推荐）

### 步骤 1：清理不需要的文件
```bash
cd ~/桌面/company-website
rm -rf node_modules dist .git
```

### 步骤 2：压缩项目文件夹
```bash
cd ~/桌面
zip -r TF-SSD-Project.zip company-website
```

### 步骤 3：对方使用方法
1. 解压 `TF-SSD-Project.zip`
2. 打开终端，进入解压后的文件夹
3. 安装依赖并构建：
```bash
npm install
npm run build
```
4. 启动服务器：
```bash
npm run preview
```
5. 打开浏览器访问：`http://localhost:4173`

---

## 方法三：创建自解压脚本（最专业）

我已经为你创建了启动脚本：

### Windows 用户
双击 `start-server.bat`

### Mac/Linux 用户
```bash
chmod +x start-server.sh
./start-server.sh
```

## 📁 最终打包内容

打包时需要包含以下文件：

```
company-website/
├── src/                    # 源代码
├── public/                 # 静态资源
├── package.json            # 项目配置
├── vite.config.js          # 构建配置
├── tsconfig.json           # TypeScript 配置
├── index.html              # 入口文件
├── start-server.sh         # Mac/Linux 启动脚本
├── start-server.bat        # Windows 启动脚本
└── 离线打包教程.md          # 本教程
```

---

## ⚠️ 注意事项

1. **对方电脑需要安装 Node.js**（方法二和三）
   - 下载地址：https://nodejs.org/
   - 推荐下载 LTS 版本

2. **Python 方法不需要安装任何软件**（方法一）
   - Mac/Linux 自带 Python
   - Windows 需要安装 Python 或使用方法二

3. **视频文件**
   - 确保 `media1.mp4` 在 `public` 文件夹中
   - 构建后会在 `dist` 文件夹中
