# TF SSD 网站 - 离线展示说明

## 🚀 快速启动（推荐）

### Windows 用户
双击运行 `start-server.bat`

### Mac/Linux 用户
在终端运行：
```bash
chmod +x start-server.sh
./start-server.sh
```

### 或者直接运行命令
```bash
npm run preview
```

启动后打开浏览器访问：**http://localhost:4173**


---

## 📦 其他离线方案

### 方案2：使用 Python 服务器
```bash
cd dist
python3 -m http.server 8080
```
然后访问：http://localhost:8080

### 方案3：使用 Node.js http-server
```bash
# 安装（只需一次）
npm install -g http-server

# 运行
cd dist
http-server -p 8080 -c-1
```

### 方案4：直接打开 HTML 文件
⚠️ 不推荐，视频可能无法播放

直接双击 `dist/index.html` 打开

---

## ⚠️ 注意事项

1. **保持终端窗口打开** - 关闭窗口服务器就停止了
2. **不要移动 dist 文件夹** - 所有资源都在里面
3. **端口被占用？** - 换个端口：`npm run preview -- --port 3000`

---

## 📁 文件说明

```
项目文件夹/
├── dist/              ← 构建后的网站（所有资源都在这里）
│   ├── index.html

│   └── media1.mp4     ← 视频文件
├── start-server.sh    ← Mac/Linux 启动脚本
├── start-server.bat   ← Windows 启动脚本
```

## 🎯 展示时建议

1. **提前测试** - 展示前先运行一次，确保一切正常
2. **准备备用方案** - 如果视频播不了，可以用下载链接
3. **全屏展示** - 按 F11 进入全屏模式更专业
4. **关闭通知** - 展示前关闭系统通知，避免打扰

---

## ❓ 常见问题

**Q: 视频播放不了？**
A: 确保 `dist/media1.mp4` 文件存在，且浏览器支持 MP4 格式

**Q: 端口被占用？**
A: 运行 `npm run preview -- --port 3000` 换个端口

**Q: 样式错乱？**
A: 确保在项目根目录运行命令，不是在 dist 目录
